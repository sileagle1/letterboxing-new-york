Inspiration

From: pblber65 | Date: 2006-04-06 16:07:48 UTC
I just got done reading the book "Weird New York". It was great. It is
a cross between a Coffee Table Book and a travel guide which will tell
you about many off beat, odd, strange, curious and even haunted
places. Some of these spots already have LB's planted near them but
there are many more that could use LB's if anybody is looking for a
place to plant. Even if you aren't looking for plants it is well worth
a read.
