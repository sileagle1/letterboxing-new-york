Hi Phynstar,

 

Are you using Microsoft office?  That’s what I paste into.

Try this:  after you paste, choose “edit”, “select all” so the whole thing is highlighted.  Then under format, choose “font”, and under font color select black.  This way all the text will turn black so you can see it.
