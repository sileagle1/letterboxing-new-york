I can answer part of that one for ya! The event stamps
can ONLY be gotten by going to the event. They're
still posted so people can record their find that day,
but if you didn't go, you won't be able to do it. :-(

travelling event stamp. Those you might find at
another gathering, so they're not entirely off the
table as yet (in fact, I accidentally picked them up
the other day, so they're currently with ME!). Not
sure where CQ is sending them next, but I have to
return them to him first.

From: Walter Quinn III | Date: 2005-09-06 02:50:08 UTC
Well, from what i've noticed, most of the Event stamps are usually posted as traditional boxes. the AQ Cermonial one was a Mystery, because the event itself was a mystery thing. As for the box of game bones, that's a traveling event stamp, but is listed under mystery, because they don't have a permenant location, they go from gathering to gathering. Some traveling event stamps are listed as such in their titles, but i guess it really depends on what the author wants to post it as. I guess most of the time, research is in order.
