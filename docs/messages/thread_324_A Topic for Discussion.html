A Topic for Discussion

From: Paul | Date: 2005-09-03 14:23:36 UTC
As we're settling in as LbNY, I have a couple of questions that I'd
like to hear about from our members. These questions might help
understand more about the group as a whole.

We recently made a fairly big jump in the number of members. At the
start of August there were 51 members, and now a month later, we
have 76.

Here's what I'd like to know more about from you:
--------------------------------------------------------------

So far, has joining this group been helpful in the sharing of
information and building a sense of community? Are we providing
something for you that LbNA isn't for example?

What can LbNY do to make letterboxing in NY even better?
--------------------------------------------------------------

I'd appreciate hearing your thoughts, and also your stories about
things that have happened as a result of joining LbCNY/LbWNY and
now, LbNY.

Thanks!
Paul in SF

RE: [LbNY] Re: A Topic for Discussion -printing clues

From: JuneMcAllister | Date: 2005-09-04 23:57:59 UTC
I've printed out more than 2-3 boxes for traveling and I still print the web page, the clues for each box, one at a time. If you do a print preview, you can see how many pages it is going to print out, see if the clues need more than one page, and if not, just set it to print one page instead of that last one that has nothing on it. And if you have a bunch of boxes that have just one page of clues, print some other boxes on the other side of the page. I zip along pretty fast that way.
 
The link he means is "Print View", at the top of each box clue page, not export. If you click on that, you get just the text of the clues, without anything else, Actually I never even noticed that before.

From: Littlejohn, Judith M | Date: 2005-09-06 13:15:40 UTC

I do the Word thing.

 

I go to the Print page for each box, then copy and paste it to a word document (right click).  That way I can fit a ton of clues on one page and not have any unnecessary graphics (like all those symbols on Atlasquest – it’s nice info to know when you’re researching, but you don’t need to carry those around in your backpack.)

 

Plus I can save the word file and edit it later if I don’t find all the boxes, and I can print on both sides of the page, and it’s just a lot easier for me.

 

But, to each his own!

Re: A Topic for Discussion - word file

From: Rick in Boca | Date: 2005-09-06 19:10:24 UTC
I end up doing this (creating a word file and printing) for some
people's web pages. Some of these web pages don't allow you to
increase the text size (it makes it easier to follow clues on the
trail if I don't have to wear my specs to read the clues, especially
when it is rainy), so I cut and paste the clues into a file and print
with a font size of 14.

The LbNA site just automatically jumps over to their web page, so you
can't print much from LbNA. I am thinking of clues by PCLG and
MartianMaggot for example.

Anyway, back on topic, I don't think this forum can help with that
particular problem. I think the main value of a regional site like
LbNY is that we may have met each other, either on the trail or at a
gathering, so we can be more familiar with each other and more
specific about some of our favorite boxes and hikes.

Rick_in_Boca (from the seventh Borough)

From: Elizabeth Hughes | Date: 2005-09-06 22:44:44 UTC
Veering off topic somewhat (....although I do the same thing a few of
the others do when I copy clues by cutting out all the extraneous
stuff, extra spaces, and enlarge the font so I can actually read
it....),

A number of my boxes in parks like Clark & Green Lakes have lengthy
driving instructions and other info that I've arranged to always
print on page one of the pdf, and have the clues begin on page two.
So if someone is doing a bunch of boxes, or groups in a series in the
same park, check the clues first before hitting the print button and
you might save yourself some ink and paper by excluding page one.

Cheers,
catbead
